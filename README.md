# DevRoast

Uma plataforma divertida de "code roasting" onde você cola seu código e recebe uma análise crítica — às vezes brutalmente honesta, às vezes em modo roast completo.

## O que é o DevRoast?

Você já quis saber o quanto seu código é... questionável? O DevRoast é a plataforma perfeita para descobrir! Basta colar seu código, escolher o modo de roast e receber uma avaliação que não poupará detalhes sobre as más práticas, códigos confusos e soluções criativas (leia-se: cuestionáveis).

## Funcionalidades

- **Input de Código**: Cole qualquer código em várias linguagens de programação
- **Modo Roast**: Ative o modo sarcasmo máximo para reviews ainda mais ácidos
- **Leaderboard da Vergonha**: Veja os códigos mais cuestionáveis da comunidade
- **Análise Visual**: Código formatado com syntax highlighting para facilitar a leitura

## Feito com

Este projeto foi construído durante o **NLW (Next Level Week)** da Rocketseat, aprendendo boas práticas de desenvolvimento web moderno.

## Como rodar

```bash
# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## Disclaimer

O DevRoast é apenas para fins de entretenimento. Nenhum código foi machucado durante a produção deste app.
